Fluxo do Sistema Condominial

1. Captura de Imagem

O sistema pode receber imagens de duas fontes:

  • Câmeras de segurança em tempo real
  • Upload manual de imagens pelo painel do porteiro

2. Processamento de Imagem

A imagem é processada para:

  • Detecção de rostos (usando OpenCV e face_recognition)
  • Reconhecimento de placas (usando EasyOCR)
  • Otimização para melhorar a qualidade

3. Reconhecimento

Dependendo do tipo de verificação:

  • Facial: Compara com os rostos cadastrados no banco de dados
  • Placa: Extrai o texto da placa e verifica no cadastro de veículos

4. Controle de Acesso

O sistema verifica:

  • Se o morador/veículo está cadastrado
  • Se há vagas disponíveis (para veículos)
  • Se está dentro do horário permitido (para visitantes)

5. Registro e Ação

Finalmente o sistema:

  • Registra a movimentação no banco de dados
  • Libera ou nega o acesso fisicamente
  • Atualiza os painéis de status
  • Notifica o síndico se necessário
